# Woodworking Planner

Plan woodworking projects with visual layouts and cut lists.

## Tech Stack

- React 19 + TypeScript (strict)
- Vite 7 (SWC)
- Tailwind CSS v4
- React Router v7
- Vitest + React Testing Library
- ESLint v9 + Prettier
- Husky + lint-staged

## Getting Started

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Run tests
pnpm test

# Lint
pnpm lint

# Format
pnpm format

# Production build
pnpm build
```
