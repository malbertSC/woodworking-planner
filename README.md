# Woodworking Planner

Plan woodworking projects with visual layouts and cut lists.

<img width="1490" height="791" alt="image" src="https://github.com/user-attachments/assets/93456815-8e39-411b-84ea-0caa02adb78b" />

## Features                                                                                                                  
                                                                                                                          
  - Gridfinity-based drawer layout — configure columns, rows, and bin heights using the 42mm Gridfinity grid standard       
  - Interactive 3D visualization — orbit, pan, and zoom a full 3D model with clickable drawers that open and close          
  - 2D technical drawings — front view, side view, and slide layout diagrams with dimension lines and tooltips              
  - Detailed cut lists — aggregated parts list grouped by material and thickness
  - Sheet layout optimization — guillotine bin-packing algorithm to minimize plywood waste across standard or custom sheet
  sizes
  - 3D-printable slide jig — generates drilling jig geometry with STL export, segmented to fit standard print beds

## Getting Started

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Run tests
pnpm test

# Lint
pnpm lint

# Format
pnpm format

# Production build
pnpm build
```
